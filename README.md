# 🐾 Pampered Paws – Pet Adoption Web Application  

## 📖 Overview  

Pampered Paws is a full-stack web application developed to simplify the pet adoption process.  
The platform allows users to explore pets available for adoption through a clean and user-friendly interface.

This project helped me strengthen my knowledge in frontend development, backend integration, and database connectivity.

---

## 🚀 Features  

- 🐶 View available pets for adoption  
- 📋 Simple and responsive user interface  
- 🔐 Backend server using Node.js and Express  
- 🗄️ MongoDB database integration  
- 📦 Structured frontend and backend architecture  

---

## ⚙️ Installation & Setup  

1. Clone the repository  
   git clone [https://github.com/your-username/your-repository-name.git](https://github.com/Devika9511/Pampered-Paws.git)  

2. Navigate to backend folder  
   cd Backend  

3. Install dependencies  
   npm install  

4. Start the server  
   node server.js  

5. Open the frontend  
   Open the HTML file in your browser.
